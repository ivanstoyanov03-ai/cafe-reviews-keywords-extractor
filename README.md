# Извлечение ключевых слов из отзывов кафе

## Описание проекта

Проект выполняет извлечение ключечевых слов из предобработанных текстов отзывов о кафе с использованием метрики TF-IDF.

## Задача

Написать функцию `find_keywords()`, которая:
- Загружает данные из CSV-файла в DataFrame
- Извлекает ключевые слова из столбца `normalized_text`
- Добавляет столбец `key_words` со списком кортежей (слово, уверенность)
- Возвращает обновленный DataFrame

## Алгоритм работы

1. **Загрузка данных** — чтение CSV файла с помощью pandas
2. **Токенизация** — разбиение текста по пробелам
3. **Фильтрация** — удаление стоп-слов, цифр, слишком коротких и длинных токенов
4. **Вычисление IDF** — inverse document frequency для всех слов в корпусе
5. **Вычисление TF-IDF** — для каждого отзыва
6. **Отбор топ-3** — выбор слов с наибольшим TF-IDF

## Формулы

- **TF (term frequency)**: `count(term) / total_terms_in_document`
- **IDF (inverse document frequency)**: `log((N + 1) / (df + 1)) + 1`
- **TF-IDF**: `TF * IDF`
